Introduction

Below is a list of my recent summary and extension in Machine Learning and Data Mining course. This course focuses mainly on theories and provides detailed mathematical foundations to understand different models. The Chinese version videos for the course have been uploaded and being updated on bilibili .

Notes

  1. The slides only cover the most important topics in my opinion in machine learning and data mining and this course focuses mainly on interviews of related jobs such as data scientists.

  2. There exist some typos on the slides. It will be better to watch videos with slides because some important details (such as proofs) are not shown on the slides...

  3. The video of the second case study will not be uploaded because it has nothing to do with data mining theories, but you could email me to ask for the recorded video of this chapter if you want.

  4. Python for Data Analysis tutorials will not be shown in this course, but mastering it is really important in real projects.
